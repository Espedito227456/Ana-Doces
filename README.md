




# Ana Doces 🍰

Site simples para uma confeitaria, apresentando produtos e interação com o usuário.

## 💡 Sobre o projeto
O Ana Doces é uma aplicação web desenvolvida para simular um site de vendas de doces, com foco em interface e interação utilizando JavaScript.

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## ⚙️ Funcionalidades
- Exibição de produtos
- Interação com botões
- Estrutura de layout responsivo

## 🎯 Objetivo
Praticar desenvolvimento front-end e manipulação do DOM.

## ▶️ Como usar
Abra o arquivo index.html no navegador
